The Conversion Process

  1. Set the background color of this page here:

  2. Install your desired theme in VS Code and switch to it

  3. Open the command palette with on mac with:

    COMMAND+SHIFT+p

    or Windows with:

    CONTROL+SHIFT+p

  4. Run the VS Code command: 

    Developer: Generate Color Theme From Current Settings

  5. Copy the resulting JSON into this field:
